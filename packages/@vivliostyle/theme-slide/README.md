# Slide

A slide theme for Vivliostyle.

## Install

```bash
npm install --save @vivliostyle/theme-slide
# or
yarn add @vivliostyle/theme-slide
```

## Use

In `vivliostyle.config.js`:

```js
module.exports = {
  theme: '@vivliostyle/theme-slide',
};
```
